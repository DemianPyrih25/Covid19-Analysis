# COVID-19 Global Analysis

A data analysis project examining the global COVID-19 pandemic from 2020 to 2025.

## Overview

This project analyzes COVID-19 data from the World Health Organization (WHO), including cases and deaths across 237 countries and territories. The analysis visualizes the pandemic's progression, identifies the most affected countries, and provides key statistics.

## Dataset

- **Source**: WHO COVID-19 Global Data
- **Records**: 500,000+ entries
- **Period**: January 2020 - 2025
- **Coverage**: 237 countries/territories worldwide

## Analysis Includes

1. **Global Case Dynamics** - Cumulative COVID-19 cases over time
2. **Top 10 Countries** - Countries with the highest number of cases
3. **Daily New Cases** - Daily infection trends showing pandemic waves
4. **Statistics** - Total cases, deaths, mortality rate, and affected countries

## Technologies Used

- Python 3.8+
- pandas - Data manipulation and analysis
- matplotlib - Data visualization
- seaborn - Statistical graphics

## Installation

```bash
pip install -r requirements.txt
```

## Usage

1. Place the `covid_data.csv` file in the same directory as the notebook
2. Open `COVID19_Analysis.ipynb` in Jupyter Notebook or JupyterLab
3. Run all cells to generate the analysis and visualizations
